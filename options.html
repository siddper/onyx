Onyx Settings

Themes

Choose a built-in theme or import a custom theme from a JSON file. Applies to the side panel editor.

    Caret

    Customize the cursor appearance and animation in the editor.

    Fonts

    Choose a preset or select “Custom” to paste a Google Fonts (or other) @import URL and font family name.

    Import into Obsidian

    Defaults used when you right-click selected text and choose “Import selection to Obsidian”. The note name and folder below are used for that action.

    Saved vaults

    Vaults you add here appear in the side panel dropdown. Choose one when exporting to Obsidian.

      Export template

      Prepended to the note when you click “Export to Obsidian”. Use {{date}}, {{time}}, and {{title}}; they are replaced with the export date, time, and note title. Leave blank for no template.

      Custom CSS

      CSS applied only to the side panel (not this Settings page). Your selectors are automatically scoped so they override default styles.

      Other

      Default vault and folder pre-fill the side panel when empty. Source count shows words/chars in the Source pane header.
